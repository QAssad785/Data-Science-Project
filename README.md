# Data-Science-Project
Welcome to my Data Science Projects Repository! This repository contains a collection of my data science projects, showcasing my skills and expertise in the field. Each project demonstrates different aspects of data analysis, machine learning, and visualization.


## **Projects:**
1. [Car Price Prediction:](https://github.com/QAssad785/Data-Science-Project/tree/main/Car%20price%20Prediction)
   
    - **Description:** The project predicts the price of the car in Poland according to their feature.
   - **Technologies Used:** The notebooks uses Decision Tree Regression, KN Neighbour Regression and Random forest Regression.
   - **Results:** The Random forrest Regression give the best value with 14% error and 93% of variance in car price.
